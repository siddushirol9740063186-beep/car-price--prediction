🚗 Car Price Prediction

📌 Project Overview
This project predicts the price of used cars using Machine Learning. It helps users estimate a fair selling price based on different car features.

📊 Dataset
The model is trained using the CarDekho dataset, which includes:

Year of manufacture
Present price
Fuel type
Seller type
Transmission
Kilometers driven

⚙️ Technologies Used

Python
Pandas
NumPy
Scikit-learn
Streamlit

🚀 Features

Easy car price prediction
Simple and user-friendly interface
Fast and accurate predictions
Real-time results using Streamlit

📂 Project Structure
car-price-prediction/
│── app.py (Streamlit app)
│── model.pkl (Trained ML model)
│── car_price_model.ipynb (Model training notebook)
│── requirements.txt (Dependencies)
│── README.md (Project documentation)

▶️ How to Run

Clone the Repository
git clone https://github.com/your-username/car-price-prediction.git

cd car-price-prediction
Install Dependencies
pip install -r requirements.txt
Run the App
streamlit run app.py

📈 Output
The model predicts the estimated price of a used car based on user inputs.
